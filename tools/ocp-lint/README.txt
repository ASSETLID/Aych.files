This directory contains the sources of `ocp-lint`:

* `utils/`: useful functions not specific to ocp-lint
* `config/`: management of configuration files
* `init/`: initial modules (global variables)
* `db/`: persistent database of warnings
* `api/`: types and modules used by lint plugins
* `output/`: printing of warnings
* `main/`: main modules

Documentation should be available in the `DOCS` directory.
