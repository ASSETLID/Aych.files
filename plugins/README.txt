This directory contains some plugins for `ocp-lint`:

* `ocp-lint-plugin-files`: analyses working on relationships between files
* `ocp-lint-plugin-text`: analyses working on source text
* `ocp-lint-plugin-parsing`: analyses working at parsing phase
* `ocp-lint-plugin-parsetree`: analyses working on parsetree (AST)
* `ocp-lint-plugin-typedtree`: analyses working on typedtree (typed AST)
* `ocp-lint-plugin-complex`: analyses working on multiple sources
* `ocp-lint-plugin-sempatch`: analyses using semantic patches
